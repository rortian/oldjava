§1.47 Class IllegalAccessError

 
public class java.lang.IllegalAccessError
    extends java.lang.IncompatibleClassChangeError (§1.48)
{
    public IllegalAccessError();	§1.47.1
    public IllegalAccessError(String s);	§1.47.2
}
The Java runtime throws an instance of this class if an application attempts to access or modify a field, or to call a method that it does not have access to.

Normally, this error is caught by the compiler; this error can only occur at runtime if the definition of a class has incompatibably changed
