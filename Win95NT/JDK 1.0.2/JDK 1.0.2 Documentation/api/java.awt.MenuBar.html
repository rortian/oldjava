This class that encapsulates the platform's concept of a menu bar bound to a Frame.

In order to attach the menu bar to a frame, the setMenuBar method (II-§1.19.28) in class Frame must be called.

addNotify


public void addNotify()




This method calls the createMenuBar method (II-§1.41.15) of this object's 
toolkit (II-§1.10.20) in order to create a MenuBarPeer  (II-§3.13) for this 
menu bar. This peer allows the application to change the look of a menu 
barwithout changing its functionality.




Most applications do not call this method directly.
